# Tic Tac Toe – JavaScript

Un jeu **Tic Tac Toe (Morpion)** simple réalisé en **HTML, CSS et JavaScript**.  
Le jeu se joue à deux joueurs directement dans le navigateur.

---

## Structure du projet

├── index.html # Structure HTML du jeu
├── index.js # Logique du jeu (JavaScript)
├── TikTakToe.css # Style et mise en page
└── README.md # Documentation du projet


---

## Fonctionnalités

- Jeu à deux joueurs (X et O)
- Détection automatique des victoires
- Gestion du match nul
- Bouton pour redémarrer la partie
- Interface simple et responsive

---

## Technologies utilisées

- **HTML5**
- **CSS3**
- **JavaScript**

---

## Comment jouer

- Ouvrir le fichier `index.html` dans un navigateur
- Cliquer sur une case pour jouer
- Le joueur gagnant est affiché à l’écran
- Cliquer sur **Restart** pour recommencer

---

## Aperçu

- Grille de jeu **3x3**
- Les joueurs jouent chacun leur tour
- Les symboles **X** et **O** s’affichent dans les cases
